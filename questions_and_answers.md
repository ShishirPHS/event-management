###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: '{}'

<i>Because here declared greeting but then mistakenly assigned an empty object to a variable named greetign. greetign was not declared with let, var, or const, it becomes a global variable (if not in strict mode), and an empty object is assigned to it. When log greetign to the console, it will print: {}</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

  if (num) count += 1;
});

```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: 3

<i>Here is a nums array. And we use the forEach loop on the array of numbers. Each loop checks array elements before updating the value of the count variable.
